# NanoHAT OLED Armbian
NanoHAT OLED and GPIO Button Control for Armbian

### Dependences
```
sudo apt install \
  libjpeg-dev \
  libfreetype6-dev \
  python \
  python-dev \
  python-pip \
  python-setuptools \
  python-smbus \
  ttf-dejavu \
  zlib1g-dev

sudo pip install \
  image \
  pillow
```

### Running
```
python nano_hat_oled.py
```
